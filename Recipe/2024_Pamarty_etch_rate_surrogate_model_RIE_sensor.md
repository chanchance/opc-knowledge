# Data-Driven Surrogate Model for Etch Rate Profiles Using Sensor Data from a Reactive Ion Etcher

## 메타데이터
- **저자**: Abhijit Pranav Pamarty, Robert Neuweiler, Le Quyen Do, Keaton Johnson, James J. Sanchez, Dinesh Koli
- **연도**: 2024
- **게재지/학회**: arXiv:2409.12925
- **DOI/URL**: https://arxiv.org/abs/2409.12925
- **인용수**: N/A (2024년 출판)

## 핵심 요약
RIE(반응성 이온 에처)의 센서 데이터를 사용하여 에칭 속도 프로파일을 예측하는 데이터 기반 서로게이트 모델이다. 오토인코더-다중 이차 보간법 하이브리드 접근법을 사용하며, 오토인코더가 에칭 속도 프로파일 특성을 잠재 공간으로 인코딩하고, 방사 기저 함수를 통해 센서 파라미터에 매핑한다. 학습 및 테스트 데이터셋 전반에 걸쳐 1% 미만의 재구성 오류를 달성한다.

## 주요 기여
- **비계측 대안**: 비용 및 시간 집약적 계측 없이 에칭 속도 프로파일 예측
- **하이브리드 아키텍처**: 딥러닝(오토인코더) + 고전적 보간법(다중 이차 RBF) 결합
- **이중 평가**: 보간 및 외삽 두 체제에서 모델 성능 분석
- **높은 정확도**: 훈련 및 테스트 데이터셋에서 1% 미만 재구성 오류

## Recipe/Process Flow 상세

### RIE 에칭 속도 예측 파이프라인

**데이터 수집**
- 17개 훈련 웨이퍼의 다중 위치 에칭 속도 측정
- 3개 센서 파라미터:
  - 에칭 가스 유량
  - 애싱 가스 유량
  - 전자기 전력 (RF 파워)

**데이터 전처리**
- 이산 측정값 → 연속 2D 스칼라 필드로 보간
- 웨이퍼 전체 에칭 속도 분포 맵 생성

**하이브리드 모델 아키텍처**

**컨볼루션 오토인코더**
- 3개 컨볼루션 레이어 (필터 크기: 5×5, 3×3, 2×2)
- 완전 연결 레이어로 잠재 공간 압축
- 맞춤형 손실 함수:
  - 재구성 오류
  - 최대 편차 패널티
  - 정규화 항

**다중 이차 RBF 매핑**
- 각 잠재 공간 변수를 센서 공간에 매핑
- 새로운 동작 조건에서 합성 에칭 프로파일 생성
- 물리적으로 합리적인 보간 제공

**예측 흐름**
1. 새 공정 파라미터 (가스 유량, 전력) 입력
2. RBF 보간으로 잠재 공간 변수 예측
3. 디코더로 전체 에칭 속도 프로파일 재구성
4. 2D 웨이퍼 에칭 속도 분포 맵 출력

### 실험 결과

**정확도 지표**
- 재구성 오류: 거의 모든 샘플에서 0.2% 미만
- 센서 예측 평균 에칭 속도 오류: 모든 테스트 케이스에서 5% 미만
- 외삽 케이스 (테스트 18): ~9% 오류 (상대적으로 높음)

**에칭 프로파일 특성**
- 전형적 웨이퍼 에칭 프로파일 포착:
  - 중심부: 낮은 에칭 속도
  - 엣지: 높은 에칭 속도
- 애싱 가스 변동에 대해 에칭 가스 변동보다 더 나은 성능

## OPC 툴 구현 관련성
- **에칭 후 OPC 모델**: RIE 에칭 속도 프로파일 예측 모델을 OPC 레시피의 에칭 바이어스 보정 컴포넌트로 통합
- **웨이퍼 위치별 에칭 바이어스**: 중심-엣지 에칭 속도 차이를 웨이퍼 위치별 OPC 레시피 보정에 반영
- **공정 파라미터 기반 레시피 조정**: 가스 유량, RF 전력 변화에 따른 에칭 바이어스를 OPC 레시피 파라미터로 자동 보정
- **잠재 공간 파라미터**: 에칭 프로파일의 잠재 공간 표현을 OPC 공정 모델의 효율적 파라미터로 활용
- **가상 계측 통합**: 실제 계측 없이 에칭 속도를 예측하는 가상 계측을 OPC 레시피 캘리브레이션 파이프라인에 통합
