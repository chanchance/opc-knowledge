# Neural Lithography: Close the Design-to-Manufacturing Gap in Computational Optics with a 'Real2Sim' Learned Photolithography Simulator

## 메타데이터
- **저자**: Cheng Zheng, Guangyuan Zhao, Peter T.C. So
- **연도**: 2023
- **게재지/학회**: SIGGRAPH Asia 2023 / arXiv:2309.17343
- **DOI/URL**: https://arxiv.org/abs/2309.17343
- **프로젝트 페이지**: https://neural-litho.github.io
- **인용수**: 다수 인용

## 핵심 요약
Neural Lithography는 계산 광학의 설계-제조 격차를 해소하기 위한 완전 미분 가능 설계 프레임워크다. 사전 학습된 포토리소그래피 시뮬레이터를 모델 기반 광학 설계 루프에 통합하고, 물리 기반 모델링과 실험 데이터 기반 훈련을 결합하는 Real2Sim 접근법을 사용한다. 홀로그래픽 광학 소자(HOE)와 다중 레벨 회절 렌즈(MDL) 설계 및 제조에서 효과를 입증한다.

## 주요 기여
- **Real2Sim 통합**: 사전 학습된 포토리소그래피 시뮬레이터를 광학 설계 루프에 직접 통합
- **완전 미분 가능 프레임워크**: 설계에서 제조까지의 전체 파이프라인 미분 가능
- **제조 가능성 정규화자**: 시뮬레이터가 제조 가능성에 대한 정규화자로 작용하여 구조 불일치 보상
- **듀얼 검증**: HOE 및 MDL 두 응용에서 설계 및 제조 성능 향상 입증

## Recipe/Process Flow 상세

### Neural Lithography 프레임워크

**문제 정의**
기존 계산 광학 설계 접근법의 한계:
- 제조 공정의 수치 모델링 무시
- 설계와 제조 광학 소자 간 성능 편차 발생
- 설계-제조 격차 (Design-to-Manufacturing Gap)

**Real2Sim 접근법**
- Real: 실험 데이터로 시뮬레이터 훈련 (실제 제조 → 시뮬레이션 보정)
- Sim: 보정된 시뮬레이터로 가상 최적화 수행
- 물리 기반 모델링 + 데이터 기반 훈련 결합

### 포토리소그래피 시뮬레이터 구조

**물리 기반 구성요소**
- 투영 광학 모델 (수차 포함)
- 레지스트 반응 모델 (두광자 리소그래피용)
- 기하학적 왜곡 모델

**데이터 기반 보정**
- 실험 수집 데이터셋으로 시뮬레이터 파라미터 최적화
- 물리 모델이 포착하지 못하는 잔여 오차 보정
- 미분 가능 설계로 그래디언트 역전파 가능

### 미분 가능 설계 최적화 루프

**단계 1: 초기 광학 설계**
- 목표 광학 기능 정의 (HOE, MDL 등)
- 초기 마스크/패턴 파라미터 설정

**단계 2: 포토리소그래피 시뮬레이션**
- 사전 학습된 시뮬레이터로 제조 결과 예측
- 패턴 변형, 에지 라운딩 등 제조 효과 시뮬레이션

**단계 3: 광학 성능 평가**
- 시뮬레이션된 제조 소자의 광학 응답 계산
- 목표 광학 기능과의 차이 계산

**단계 4: 그래디언트 기반 최적화**
- 광학 손실 → 제조 시뮬레이터 → 마스크 파라미터로 역전파
- 제조 가능성을 고려한 설계 업데이트

**단계 5: 수렴 후 마스크 생성**
- 최적화된 설계에서 제조 마스크 생성
- 두광자 리소그래피 시스템으로 제조

### 응용 사례

**HOE (홀로그래픽 광학 소자)**
- 복잡한 홀로그래픽 기능을 수행하는 광학 소자
- Neural Lithography로 제조 후 설계 성능에 근접

**MDL (다중 레벨 회절 렌즈)**
- 여러 회절 레벨을 갖는 렌즈 구조
- 제조 유도 왜곡 사전 보상으로 성능 향상

## OPC 툴 구현 관련성
- **리소그래피 시뮬레이터 통합**: Neural Lithography의 Real2Sim 접근법을 OPC 레시피의 공정 모델 캘리브레이션에 적용
- **미분 가능 OPC 루프**: 포토리소그래피 시뮬레이터를 OPC 최적화 루프에 직접 통합하는 완전 미분 가능 파이프라인
- **물리+데이터 하이브리드 모델**: OPC 레시피의 리소그래피 모델로 물리 기반 + 데이터 기반 하이브리드 시뮬레이터 구현
- **두광자 리소그래피 OPC**: 반도체 OPC 외 두광자 리소그래피 시스템에 대한 OPC 레시피 설계 가능성
- **설계-제조 격차 해소**: OPC 레시피의 궁극적 목표인 설계-제조 격차 최소화를 위한 프레임워크 참조
