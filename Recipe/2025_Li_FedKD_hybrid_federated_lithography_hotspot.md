# FedKD-hybrid: Federated Hybrid Knowledge Distillation for Lithography Hotspot Detection

## 메타데이터
- **저자**: Yuqi Li, Xingyou Lin, Kai Zhang, Chuanguang Yang, Zhongliang Guo, Jianping Gou, Yanli Li
- **연도**: 2025
- **게재지/학회**: arXiv:2501.04066
- **DOI/URL**: https://arxiv.org/abs/2501.04066
- **인용수**: N/A (2025년 출판)

## 핵심 요약
FedKD-hybrid는 분산 반도체 제조 환경에서 프라이버시를 보존하면서 리소그래피 핫스팟 감지를 위한 연합 학습 프레임워크다. 파라미터 기반 집계와 지식 증류를 결합하여 더 풍부한 지식 전이를 달성한다. ICCAD-2012 및 실세계 FAB 데이터셋에서 최신 FL 방법 대비 최대 33.05% 정확도 향상을 달성하며 반도체 IP 보호 규제를 준수한다.

## 주요 기여
- **하이브리드 지식 증류**: 파라미터와 로짓 예측 모두를 활용한 이중 채널 지식 전이
- **FedKD-hybrid 알고리즘**: 공개 데이터셋으로 글로벌 합의를 달성하는 연합 학습 프레임워크
- **이종 FL 지원**: 참여자별 다른 모델 아키텍처 허용 (유연한 구현)
- **우수한 성능**: ICCAD-2012에서 1.45-8.19%, FAB에서 1.85-33.05% 정확도 향상

## Recipe/Process Flow 상세

### 반복당 3단계 학습 프로토콜

**단계 0: 동일 레이어 합의**
- 클라이언트 간 동일하게 유지할 네트워크 레이어 지정 (보통 첫 번째 컨볼루션 레이어, 완전 연결 레이어)
- 파라미터 기반 집계를 가능하게 하는 최소 공유 구조
- 중간 레이어는 클라이언트별 독자 아키텍처 유지

**단계 I: 로컬 훈련 및 로짓 생성**
- 각 참여자가 사적 데이터로 이종 모델 훈련
- 공개 데이터셋(ICCAD-2012)에서 모델 평가 → 출력 로짓 생성
- 동일 레이어 파라미터와 로짓을 서버로 전송

**단계 II: 서버 집계**
- 파라미터 집계: w̄ = (1/N)Σwᵢᵗ
- 로짓 집계: f̄(D) = (1/N)Σfᵢᵗ(D)

**단계 III: 로컬 모델 업데이트/지식 전이**
결합 손실 함수로 재훈련:
F = (1/N)Σ[ℓ(w̄, wᵢᵖ) + λᵢ||fᵢ(D) - f̄(D)||]
- λ = 0.5 (파라미터 기반과 지식 증류 균형)

### 2단계 로컬 훈련 프로세스
1. E₁ 에폭: 집계된 파라미터로 초기화하여 공개 데이터셋 훈련
2. E₂ 에폭: 사적 데이터셋으로 훈련 (로컬 특화 유지)

### 동일 레이어 전략
- Conv1(첫 번째 컨볼루션)과 FC2/3(완전 연결 레이어) 공유
- 통신 효율성과 지식 공유의 균형
- 클라이언트별 독자 중간 레이어로 유연성 유지

### 실험 결과

## OPC 툴 구현 관련성
- **분산 OPC 모델 학습**: 여러 파운드리가 사적 데이터를 공유하지 않고 공동으로 OPC 핫스팟 감지 모델 학습
- **IP 보호 협업**: EAR/EECM 규제를 준수하면서 반도체 IP를 보호하는 OPC 레시피 공동 개발
- **이종 아키텍처 지원**: 서로 다른 OPC 도구 구현을 사용하는 파운드리 간 지식 공유
- **핫스팟 감지 연합 학습**: OPC 레시피 실행 후 핫스팟 감지 모델을 여러 파운드리 데이터로 분산 학습
- **실세계 FAB 데이터 검증**: 실제 파운드리 데이터(FAB)에서 검증된 연합 학습 프레임워크 신뢰성
