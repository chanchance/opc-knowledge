# SegSEM: Enabling and Enhancing SAM2 for SEM Contour Extraction

## 메타데이터
- **저자**: Da Chen, Guangyu Hu, Kaihong Xu, Kaichao Liang, Songjiang Li, Wei Yang, XiangYu Wen, Mingxuan Yuan
- **연도**: 2026
- **게재지/학회**: arXiv:2602.20471
- **DOI/URL**: https://arxiv.org/abs/2602.20471
- **인용수**: N/A (2026년 출판)

## 핵심 요약
SegSEM은 OPC 모델 캘리브레이션을 위한 SEM(주사전자현미경) 이미지에서 고충실도 2D 윤곽을 추출하는 프레임워크다. SAM2(Segment Anything Model 2)를 최소한의 레이블 데이터(60개 이미지)로 반도체 SEM 도메인에 적응시키며, 신뢰도 기반 폴백 메커니즘을 통해 생산 환경에서의 시스템 신뢰성을 보장한다. 전통적 방법 대비 평균 IoU 13.07% 향상을 달성한다.

## 주요 기여
- **데이터 효율적 SAM2 적응**: 60개 생산 SEM 이미지만으로 SAM2를 반도체 SEM 도메인에 효과적 적응
- **선택적 파인튜닝 전략**: 마스크 디코더 동결 + 비전 인코더만 파인튜닝
- **신뢰도 기반 폴백 아키텍처**: SAM2 출력 품질 불충분 시 전통적 알고리즘으로 자동 전환
- **EPE 패널티 손실 함수**: OPC 캘리브레이션에 직접 연관된 EPE 기반 학습

## Recipe/Process Flow 상세

### OPC 캘리브레이션 흐름에서의 역할
1. SEM 이미지 수집 (다양한 노출 조건)
2. SegSEM으로 고충실도 웨이퍼 윤곽 추출
3. 추출된 윤곽으로 OPC 모델 캘리브레이션 데이터 생성
4. 캘리브레이션된 OPC 모델로 레시피 최적화

### SegSEM 이중 경로 파이프라인

**경로 1: SAM2 기반 처리 (주 경로)**
1. SEM 이미지 입력
2. 파인튜닝된 SAM2로 마스크 예측
3. 예측 IoU 점수 기반 신뢰도 평가
4. 품질 임계값 통과 시 형태학적 후처리

**경로 2: 전통적 알고리즘 (폴백)**
1. SAM2 출력이 품질 검사 실패 시 자동 활성화
2. Sauvola의 적응형 임계값 처리 적용
3. 극단적 노출 조건에서 안정적 윤곽 추출 보장

### SAM2 적응 전략 (선택적 파인튜닝)
- **동결**: 사전 학습된 마스크 디코더 (이미 강한 일반화 능력)
- **파인튜닝**: 이미지 인코더 + 프롬프트 인코더 (도메인 적응 필요)
- Ground-truth 마스크에서 랜덤 샘플링된 단일 포인트 프롬프트 사용
- 전처리 없이 학습 (정확한 CD 측정을 위한 노이즈 패턴 보존)

### 맞춤형 손실 함수
표준 분할 손실 + 3가지 OPC 특화 항:
1. **마스크 IoU 보상**: 예측-실측 중첩 장려
2. **마스크 수 패널티**: 허위 파편 억제
3. **EPE 패널티**: 경계 편차 패널티 (CD 측정 정확도에 중요)

### 3단계 신뢰도 기반 품질 평가
1. **신뢰도 검사**: 예측 IoU 점수 > τ_conf (예: 0.90)
2. **위상 검사**: 연결된 구성요소 분석으로 단일 연속 영역 확인
3. **기하학 검사**: 면적 및 종횡비가 설계 기반 허용 범위 내

품질 검사 실패 시 Sauvola 적응형 임계값 폴백 활성화

### 훈련 세부 사항
- **데이터셋**: 60개 생산 SEM 이미지 (50 훈련, 10 테스트)
- **하드웨어**: Ascend 910B GPU
- **옵티마이저**: AdamW (lr=1×10⁻⁵)
- **조기 종료**: 과적합 방지를 위해 1,000 반복 미만

### 평가 결과
- **평균 IoU**: 0.884 (전통적 방법 대비 13.07% 향상)
- **F1 점수**: 0.941 (7.58% 향상)
- **폴백 활성화율**: 500개 중 36개 (7.2%), 주로 극단적 노출 조건
- SAM2 기반이 92%의 경우 성공적으로 처리

## OPC 툴 구현 관련성
- **OPC 모델 캘리브레이션 파이프라인**: SEM 이미지에서 자동으로 윤곽을 추출하여 OPC 레시피 캘리브레이션 데이터 생성
- **웨이퍼 측정 자동화**: OPC 레시피 검증 사이클에서 SEM 측정 결과 자동 처리
- **EPE 기반 학습**: CD 측정 정확도와 직결되는 EPE 패널티를 OPC 캘리브레이션 품질 지표로 활용
- **신뢰도 기반 품질 보증**: OPC 레시피 캘리브레이션 데이터의 품질을 자동으로 보증하는 파이프라인
- **소수 샘플 적응**: 새 파운드리/공정에 대해 소수의 SEM 이미지만으로 OPC 측정 모델 적응
