# DevelSet: Deep Neural Level Set for Instant Mask Optimization

## 메타데이터
- **저자**: Guojin Chen, Ziyang Yu, Hongduo Liu, Yuzhe Ma, Bei Yu
- **연도**: 2023 (ICCAD21 발표)
- **게재지/학회**: ICCAD 2021 / arXiv:2303.12529
- **DOI/URL**: https://arxiv.org/abs/2303.12529
- **인용수**: 다수 인용 (핵심 OPC 논문)

## 핵심 요약
DevelSet은 GPU 가속과 딥러닝을 결합한 레벨 셋 기반 OPC 프레임워크로 금속층에서 즉각적 마스크 최적화(약 1초)를 달성한다. 마스크 복잡도를 줄이기 위한 곡률 항 도입과 GPU 가속 구현으로 계산 병목을 해결하며, 레벨 셋 고유 원칙으로 세밀하게 설계된 DNN과 GPU 가속 레벨 셋 최적화기의 공동 최적화를 실현한다.

## 주요 기여
- **향상된 레벨 셋 알고리즘**: 마스크 복잡도 감소를 위한 곡률 항 도입
- **GPU 가속**: 반복적 최적화의 계산 병목 GPU 구현으로 해결
- **DNN 아키텍처**: 레벨 셋 고유 원칙으로 설계된 딥러닝 모델
- **즉각 수준 실행**: 약 1초의 마스크 최적화 시간 달성

## Recipe/Process Flow 상세

### 레벨 셋 기반 ILT 공식화

**마스크 표현**
- 레벨 셋 함수 φ(x,y): 부호 있는 거리 함수
  - φ > 0: 마스크 불투명 영역 (크롬)
  - φ < 0: 마스크 투명 영역 (유리)
  - φ = 0: 마스크 경계 (최적화 대상)

**최적화 목표 함수**
L(φ) = L_print(φ) + λ_c · L_curvature(φ)

- L_print: 인쇄 충실도 항 (목표 패턴과의 레지스트 윤곽 일치)
- L_curvature: 곡률 정규화 항 (마스크 복잡도 제어)
- λ_c: 곡률 가중치 파라미터

### DevelSet 최적화 파이프라인

**단계 1: 레벨 셋 초기화**
- 목표 패턴으로부터 초기 레벨 셋 함수 φ_0 생성
- 부호 있는 거리 변환 (SDF) 적용

**단계 2: DNN 기반 레벨 셋 갱신 예측**
- DNN 입력: 현재 φ_t, 목표 패턴 Z*, 리소그래피 시뮬레이션 결과
- DNN 출력: 레벨 셋 갱신량 Δφ
- GPU 병렬 처리로 빠른 DNN 추론

**단계 3: GPU 가속 레벨 셋 최적화**
- φ_{t+1} = φ_t + α · Δφ
- 곡률 항으로 마스크 경계 부드러움 유지
- GPU 병렬 그래디언트 계산

**단계 4: 마스크 이진화**
- 최종 φ 함수에서 이진 마스크 생성
- φ > 0: 불투명, φ < 0: 투명
- MRC 검사 적용

### 곡률 항의 역할
- 레벨 셋 최적화에서 과도한 마스크 복잡도 방지
- Shot count 감소 (제조 비용 절감)
- 마스크 경계의 물리적 제조 가능성 향상

### GPU 가속 전략
- 레벨 셋 함수를 2D 그리드로 표현, GPU 텐서 연산 활용
- DNN과 레벨 셋 최적화 모두 GPU에서 실행
- 배치 처리로 여러 패턴 동시 최적화

### 성능 결과 (금속층)
- 실행 시간: ~1초 (즉각 수준)
- 인쇄 가능성: 기존 방법 능가
- 마스크 복잡도: 곡률 항으로 감소

## OPC 툴 구현 관련성
- **레벨 셋 OPC 모듈**: SKOPC Modeling 컴포넌트에서 레벨 셋 기반 ILT 구현에 DevelSet 알고리즘 참조
- **곡률 정규화 레시피**: OPC 레시피 파라미터로 마스크 복잡도-품질 트레이드오프 조절하는 곡률 가중치 λ_c
- **GPU 가속 ILT**: GPU 가속 레벨 셋 최적화를 OPC 레시피 실행 엔진에 통합
- **MRC 통합**: 레벨 셋 최적화 후 마스크 규칙 검사를 OPC 레시피 파이프라인에 통합
- **1초 실행 목표**: 고속 OPC 레시피 실행을 위한 DNN+레벨 셋 하이브리드 접근법
