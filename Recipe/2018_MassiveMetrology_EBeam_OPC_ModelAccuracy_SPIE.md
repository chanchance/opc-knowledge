# Massive Metrology Using Fast E-Beam Technology Improves OPC Model Accuracy by >2x at Faster Turnaround Time

## 메타데이터
- **저자**: (imec + KLA 협력팀, Zhao 외)
- **연도**: 2018
- **게재지**: Proc. SPIE 10585, Metrology, Inspection, and Process Control for Microlithography XXXII, 105852Q
- **DOI/URL**: https://doi.org/10.1117/12.2299971
- **인용수**: ~30

## 핵심 요약
대량 계측(massive metrology)을 위해 고속 e-빔 기술을 활용하여 OPC 모델 정확도를 2배 이상 향상시키고 모델 구축 TAT(Turn-Around Time)를 단축하는 방법을 제시한다. 기존 CD-SEM의 처리량 병목을 해결하는 고속 e-빔 계측 시스템(7,000 이미지/시간 이상)과 대용량 데이터를 활용하는 새로운 계산 소프트웨어를 결합하여 계계통적/랜덤 계측 오차를 크게 줄이고 OPC 모델 품질을 혁신적으로 향상시킨다.

## 주요 기여
1. OPC 모델 정확도 2배 이상 향상 (대량 계측 데이터 활용)
2. 7,000 이미지/시간 이상의 고속 e-빔 계측 시스템
3. 계통적/랜덤 SEM 계측 오차 대폭 감소
4. OPC 모델 구축 TAT 단축 (계측 병목 해소)

## Recipe/Process Flow 상세

### CD-SEM 계측 병목 문제
```
기존 OPC 모델 캘리브레이션:
  계측 단계: CD-SEM으로 게이지 패턴 측정
  기존 CD-SEM 처리량:
    수백~수천 이미지/시간
    게이지 수 증가 → 계측 시간 선형 증가
  병목:
    충분한 게이지 측정: 수일~수주 소요
    게이지 부족: 모델 정확도 한계
    계측 노이즈: SEM 이미지 노이즈 → 측정 오차

OPC 모델 정확도와 게이지 수의 관계:
  게이지 수 증가 → 더 많은 패턴 커버
  → 모델 외삽 오차 감소
  → 실제 칩 패턴 포함 확률 증가
  → 전체 OPC 정확도 향상

대량 계측의 필요성:
  10nm 이하: 복잡한 2D 패턴 증가
  2D 패턴 게이지 수: 기존 대비 수십~수백 배 필요
  → 고속 계측 시스템 필수
```

### 고속 e-빔 계측 시스템
```
기존 CD-SEM vs. 고속 e-빔:
  기존 CD-SEM:
    처리량: ~500-1000 이미지/시간
    전자 도즈: 높음 (SEM 수축 발생)
    계측 정밀도: 중간

  고속 e-빔 (KLA):
    처리량: >7,000 이미지/시간 (7× 이상 빠름)
    설계: 고 데이터율 + 대 프로브 전류 + 초저 노이즈 전자 광학
    전자 도즈: 낮음 → SEM 수축 감소
    계측 정밀도: 향상 (노이즈 감소)

처리량 향상 원리:
  병렬 e-빔: 다수 빔 동시 스캔
  최적화 이미징: 고속 + 고품질 SEM 이미지
  소프트웨어 파이프라인: 이미지 처리 자동화
```

### 대량 계측 데이터 활용
```
계측 데이터 증가 효과:
  게이지 수: 기존 수백 → 수천~수만 개
  패턴 다양성: 더 많은 CD/pitch/2D 패턴 커버
  통계 신뢰도: 대량 데이터 → SEM 노이즈 통계 평균화

계통적 오차 감소:
  동일 패턴 반복 측정: 계통 오차 정확한 정량화
  오차 보정: 계통 SEM 오차 → 데이터 보정
  효과: 실제 CD 반영도 향상

랜덤 오차 감소:
  대량 측정: 통계 평균 → 측정 불확실도 감소
  σ_random ∝ 1/√N (N: 반복 측정 수)
  N 증가 → 측정 불확실도 감소

모델 캘리브레이션 개선:
  더 많은 게이지 + 더 낮은 계측 오차
  → OPC 모델 파라미터 최적화 품질 향상
  → RMS EPE: 2배 이상 감소
```

### 새로운 계산 소프트웨어 솔루션
```
대용량 데이터 처리:
  수천~수만 SEM 이미지 자동 처리
  CD 추출: 자동 컨투어 분석
  데이터 품질 필터링: 노이즈/이상치 자동 제거

계측 오차 모델링:
  SEM 노이즈 모델: 이미지 노이즈 → CD 오차 변환
  계통 오차 모델: 패턴 의존 계통 오차 보정
  불확실도 전파: 계측 오차 → 모델 오차 기여 분리

OPC 모델 통합:
  대량 계측 → 직접 OPC 모델 캘리브레이션
  모델 검증: 대량 독립 측정 → 강건한 검증
```

### 성능 결과
```
OPC 모델 정확도:
  기존 계측 기반 모델: 기준
  대량 e-빔 계측 모델: >2× RMS EPE 개선

TAT:
  7,000 이미지/시간: 기존 대비 7× 빠른 데이터 수집
  전체 캘리브레이션 사이클: 대폭 단축

계측 품질:
  계통 오차: 대량 데이터로 정확 정량화 + 보정
  랜덤 오차: 통계 평균으로 감소
  → 모델 입력 데이터 품질 향상
```

## OPC 툴 구현 관련성
- **SKOPC 대량 계측**: `skopc/modeling/` 대량 e-빔 계측 데이터 처리 파이프라인
- **계측 오차 모델**: SEM 계측 오차를 OPC 모델 캘리브레이션에 반영
- **게이지 관리**: 대량 게이지 데이터 자동 품질 필터링 + OPC 모델 입력 최적화
- **모델 정확도 향상**: 대량 계측으로 SKOPC 컴팩트 모델 정확도 2× 개선 목표
