# AdaOPC: A Self-Adaptive Mask Optimization Framework For Real Design Patterns

## 메타데이터
- **저자**: Wenqian Zhao, Xufeng Yao, Ziyang Yu, Guojin Chen, Yuzhe Ma, Bei Yu, Martin D.F. Wong
- **연도**: 2023
- **게재지/학회**: arXiv preprint (cs.CV), arXiv:2303.12723 / ICCAD 2023 (추정)
- **DOI/URL**: https://arxiv.org/abs/2303.12723
- **인용수**: 20+

## 핵심 요약
AdaOPC는 실제 설계 패턴의 두 가지 핵심 특성--(1) 하위 영역마다 다른 패턴 복잡도, (2) 설계 레이아웃에서 동일 패턴의 반복 출현--을 활용하는 자기 적응형 마스크 최적화 프레임워크다. 패턴 복잡도에 따라 최적 OPC 솔버를 동적으로 선택하고, 반복 패턴에 대해서는 그래프 기반 동적 패턴 라이브러리에서 최적화된 마스크를 재사용하여 속도와 정확도를 동시에 최적화한다.

## 주요 기여
- **적응적 솔버 선택**: 패턴 복잡도에 따라 솔버 풀에서 최적 OPC 솔버 동적 선택
- **동적 패턴 라이브러리**: 반복 패턴의 그래프 기반 인덱스 및 마스크 재사용 시스템
- **속도/정확도 공동 최적화**: 복잡도 인식 솔버 라우팅으로 효율-품질 균형
- **실용적 설계 패턴 처리**: 학술용 합성 패턴이 아닌 실제 설계 패턴 최적화

## Recipe/Process Flow 상세

### AdaOPC 자기 적응형 OPC 플로우

```
풀칩 레이아웃 입력
        ↓
[패턴 분류 및 복잡도 측정]
  각 레이아웃 타일에 대해:
  - 패턴 복잡도 점수 계산
    (엣지 수, 2D 특성, 좁은 간격 등)
  - 복잡도 레벨 분류:
    Level 1 (단순): 1D 패턴, 규칙적 반복
    Level 2 (중간): 부분 2D, 라인엔드
    Level 3 (복잡): 완전 2D, 불규칙, 좁은 간격
        ↓
[그래프 기반 패턴 매칭]
  동적 패턴 라이브러리 검색:
  - 현재 패턴을 그래프로 인코딩
  - 기존 라이브러리 패턴과 유사도 계산
  - 유사도 > 임계값: 저장된 마스크 재사용
  - 유사도 < 임계값: 새로 최적화 후 라이브러리 추가
        ↓
[적응적 솔버 선택]
  확장 가능한 솔버 풀:
  - 솔버 A (고속/저정확도): 단순 패턴용
    예: 규칙 기반 OPC, 빠른 GAN
  - 솔버 B (중간): 중간 복잡도용
    예: 수치 EBOPC, Neural-ILT
  - 솔버 C (고정확도/저속): 복잡 패턴용
    예: 수치 ILT, DiffOPC

  선택 정책:
  복잡도 레벨 → 솔버 라우팅 테이블 → 최적 솔버
        ↓
[OPC 최적화 실행]
  선택된 솔버로 마스크 최적화
        ↓
[결과 저장]
  최적화된 마스크 → 동적 패턴 라이브러리 추가
  (유사 패턴에서 재사용 가능)
        ↓
풀칩 최적화 마스크 출력
```

### 그래프 기반 동적 패턴 라이브러리

패턴을 그래프로 표현:
- **노드(Node)**: 레이아웃의 각 폴리곤 또는 특징 포인트
- **엣지(Edge)**: 인접 폴리곤 간의 기하학적 관계 (거리, 방향)

패턴 유사도 계산:
$$sim(G_1, G_2) = \text{Graph Edit Distance}^{-1}(G_1, G_2)$$

라이브러리 관리:
- LRU(Least Recently Used) 캐시 정책
- 패턴 클러스터링으로 저장 공간 최적화
- 점진적 업데이트 (새 패턴 발견 시 자동 추가)

### 패턴 복잡도 측정 지표
1. **엣지 밀도**: 단위 면적당 폴리곤 엣지 수
2. **2D 특성 비율**: 라인엔드, 코너 등 2D 패턴 비율
3. **최소 간격**: 최소 폴리곤 간 거리 (좁을수록 복잡)
4. **대칭성**: 패턴의 기하학적 대칭 정도
5. **EPE 히스토리**: 유사 패턴에서의 과거 OPC 어려움 기록

### 성능 결과
- 기존 단일 솔버 방식 대비 속도와 성능 모두 향상
- 반복 패턴 재사용으로 전체 처리 시간 대폭 단축
- 복잡 패턴에서는 고품질 솔버로 최고 수준 마스크 품질 유지

## OPC 툴 구현 관련성
AdaOPC의 적응형 솔버 선택은 SKOPC 레시피 러너에 직접 적용 가능:
- **레이어별 솔버 라우팅**: SKOPC의 rule_opc.py와 model_opc.py를 패턴 복잡도에 따라 동적 선택
- **패턴 라이브러리**: SKOPC에 최적화 마스크 캐시 시스템 구현으로 반복 패턴 처리 가속
- **복잡도 분류**: GDS 레이아웃에서 패턴 복잡도 자동 측정하여 레시피 선택에 활용
- **확장 가능 솔버 풀**: SKOPC 아키텍처에 플러그인 방식 OPC 솔버 확장 구조 추가
