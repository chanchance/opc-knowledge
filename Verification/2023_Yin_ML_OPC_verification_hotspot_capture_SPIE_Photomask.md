# Machine Learning Assisted Effective OPC Verification Hotspot Capture

## 메타데이터
- **저자**: (Siemens EDA Calibre 팀)
- **연도**: 2023
- **게재지**: Proc. SPIE 12751, Photomask Technology + EUV Lithography 2023
- **DOI/URL**: https://doi.org/10.1117/12.2687752

## 핵심 요약
Siemens EDA Calibre 머신러닝 지원 OPC 검증 툴을 활용하여 브릿징(bridging)과 핀칭(pinching) 등 다양한 실패 메커니즘에 대한 ML 분류기를 사용해 핫스팟을 효과적으로 포착하는 방법론을 제시한다. OPC 검증 레시피 설정과 엔지니어링 작업량을 단순화하면서 핫스팟 포착 효과를 향상시키는 산업용 솔루션이다.

## 주요 기여
1. Calibre ML 기반 OPC 검증 툴에서 실패 메커니즘별(브릿징, 핀칭) ML 분류기 적용
2. OPC 검증 레시피 설정 복잡도 감소 + 엔지니어링 작업량 단순화
3. 핫스팟 포착 효과성(hotspot capture effectiveness) 향상 실증
4. 여러 실패 메커니즘을 단일 ML 프레임워크에서 통합 처리하는 방법
5. 산업용 OPC 검증 플로우에 ML 통합의 실제 구현 사례

## 검증 방법론
- **툴**: Siemens EDA Calibre 머신러닝 지원 OPC 검증
- **실패 메커니즘**: 브릿징(Bridging), 핀칭(Pinching)
- **ML 방법**: 실패 메커니즘별 ML 분류기
- **평가 지표**: 핫스팟 포착률(hotspot capture rate), 레시피 설정 복잡도
- **적용 환경**: 산업용 고급 노드 OPC 검증

## OPC 툴 구현 관련성
- Calibre 기반 OPC 검증 파이프라인에 ML 분류기 통합 실무 방법
- 브릿징/핀칭 각 실패 모드별 특화된 ML 모델 구성 전략
- OPC 검증 레시피 자동화 및 엔지니어링 효율화 방법론
- ML 지원 OPC 검증의 산업 적용 기준 사례 (Calibre 플랫폼)
