# Interlayer 3D-EPE Analysis Using Contour Distance from Design with High Landing Energy SEM Imaging on Advanced Logic Devices

## 메타데이터
- **저자**: Noyeong Chung, Gwangseok Maeng, Insung Kim, Nivea Schuch, Charles Valade, Antoine Legrain, Frederic Robert, Thiago Figueiro, Jeong-Ho Yeo, Noam Oved, Uri Smolyan, You Jin Kim, Michael Shifrin
- **연도**: 2024
- **게재지**: Proc. SPIE 12955, Metrology, Inspection, and Process Control XXXVIII, 129550N
- **DOI/URL**: https://doi.org/10.1117/12.3010831
- **소속**: Samsung Electronics (한국), Applied Materials (프랑스/이스라엘)

## 핵심 요약
고착지에너지(high landing energy) SEM 이미징을 이용해 설계(design)와의 콘투어 거리 기반으로 다층간(interlayer) 3D-EPE를 분석하는 방법론을 제시한다. EPE는 CD, 오버레이, LER을 여러 레이어에서 측정해 도출되는 복합 지표로, CMOS 스케일링에 따른 공정 복잡도 증가로 패터닝 제어의 핵심 메트릭으로 자리잡았다. 삼성전자와 Applied Materials의 협력 연구로, 고급 로직 디바이스에서 실측 콘투어-설계 거리를 통한 3D EPE 정량화 방법을 실증한다.

## 주요 기여
1. 고착지에너지 SEM을 이용한 다층 콘투어 동시 이미징 방법론
2. 설계(GDS)와의 콘투어 거리 기반 3D-EPE 측정 체계 구축
3. CD + 오버레이 + LER 통합한 다층간 EPE 분석 파이프라인
4. 고급 로직 디바이스(삼성전자 첨단 노드)에서 실증
5. 인라인 EPE 모니터링을 위한 고에너지 SEM 계측 방법론

## 검증 방법론
- **SEM 방법**: 고착지에너지(high landing energy) CD-SEM 이미징
- **콘투어 측정**: 다층 레이어 동시 이미징 → 레이어별 콘투어 추출
- **EPE 계산**: 콘투어-설계 거리 측정 → CD/오버레이/LER 기여분 분리
- **적용 디바이스**: 삼성전자 고급 로직 디바이스
- **3D 분석**: 인접 레이어 간 EPE 관계 분석 (단층 → 다층 확장)

## OPC 툴 구현 관련성
- OPC 검증 후 다층 EPE 측정의 실무 방법론 - 단일 레이어 OPC 검증의 한계 보완
- 고에너지 SEM 콘투어를 OPC 모델 검증 입력 데이터로 활용하는 방법
- 3D EPE를 OPC 성능 지표로 활용하는 다층 OPC 검증 체계 구축 참조
- 설계-콘투어 거리 기반 EPE 분석을 OPC EPE 예측과 비교 검증하는 프레임워크
