# Applications of Large Field of View E-beam Metrology to Contour-Based Optical Proximity Correction Modeling

## 메타데이터
- **저자**: Chih-I Wei, Seulki Kang, Sayantan Das, Masahiro Oya, Yosuke Okamoto, Kotaro Maruyama, Germain Fenger, Azat Latypov, Ir Kusnadi, Gurdaman Khaira, Yuichiro Yamazaki, Werner Gillijns, Sandip Halder, Gian Lorusso
- **연도**: 2023
- **게재지**: Journal of Micro/Nanopatterning, Materials, and Metrology (JMM), Vol. 22, Issue 4, Article 041603
- **DOI/URL**: https://doi.org/10.1117/1.JMM.22.4.041603

## 핵심 요약
복잡한 2D 패턴에 대해 기존 컷라인(cutline) 기반 OPC 모델 캘리브레이션의 정확도 한계를 극복하기 위해 대시야각(Large Field of View, LFoV) e-빔 계측의 이미지 콘투어를 활용한 OPC 모델 캘리브레이션 및 검증 방법론을 제시한다. 브릿징 핫스팟을 포함한 웨이퍼 샘플링 계획을 통해 핫스팟 인식 3D 레지스트(R3D) 컴팩트 모델을 생성하고, 컷라인 기반과 콘투어 기반 OPC 모델의 정확도를 비교 평가한다.

## 주요 기여
1. LFoV e-빔 메트롤로지 이미지 콘투어를 OPC 모델 캘리브레이션에 직접 활용하는 방법론
2. 컷라인 기반 vs 콘투어 기반 OPC 모델 정확도 비교 - 2D 핫스팟 패턴에서 콘투어 기반이 우수함을 실증
3. 브릿징 핫스팟 인식 웨이퍼 샘플링 계획 설계 방법
4. 핫스팟 인식 R3D(3D Resist) 컴팩트 모델 생성 파이프라인
5. LFoV 콘투어 메트롤로지의 OPC 모델 검증 적용 사례

## 검증 방법론
- **메트롤로지**: LFoV e-빔 계측 (ADI: After-Development Inspection) 이미지 콘투어
- **모델 비교**: 컷라인 기반 OPC 모델 vs 콘투어 기반 OPC 모델
- **대상 패턴**: 일반 패턴 + 브릿징 핫스팟 패턴
- **모델 유형**: 핫스팟 인식 R3D(3D Resist) 컴팩트 모델
- **평가 지표**: 2D 형상 예측 정확도, 웨이퍼 핫스팟 예측 성능
- **결론**: 콘투어 기반 모델이 2D 형상 및 핫스팟 예측에서 컷라인 기반보다 더 나은 모델 예측 제공

## OPC 툴 구현 관련성
- OPC 모델 캘리브레이션 시 2D 패턴에 LFoV e-빔 콘투어 데이터 입력의 필요성 및 방법론
- 브릿징 핫스팟 인식 샘플링 전략이 OPC 모델 품질 향상에 핵심적
- R3D 컴팩트 모델 생성에 콘투어 메트롤로지 활용하는 실무 파이프라인
- OPC 검증 단계에서 LFoV 콘투어를 이용한 핫스팟 포착률 향상 가능성
