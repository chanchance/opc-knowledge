# Lithography Hotspot Detection via Heterogeneous Federated Learning with Local Adaptation

## 메타데이터
- **저자**: Xuezhong Lin, Jingyu Pan, Jinming Xu, Yiran Chen, Cheng Zhuo
- **연도**: 2021
- **게재지/학회**: arXiv preprint (cs.LG), ICCAD 2021
- **DOI/URL**: https://arxiv.org/abs/2107.04367
- **인용수**: 미확인

## 핵심 요약
반도체 설계 하우스들이 독점 레이아웃 데이터 공유를 꺼리는 현실적 제약을 극복하기 위해, 이 논문은 연합 학습(Federated Learning) 기반의 리소그래피 핫스팟 검출 프레임워크를 제안한다. 이종(heterogeneous) 연합 학습 구조를 통해 각 설계 하우스의 로컬 데이터를 공개하지 않으면서도 글로벌 핫스팟 검출 모델을 학습한다. 비독립-동일분포(non-IID) 데이터 문제를 해결하기 위해 글로벌 서브모델과 로컬 적응 메커니즘을 결합한다.

## 주요 기여
- **이종 연합 학습 프레임워크**: 서로 다른 설계 하우스 간 이종 데이터를 처리하는 연합 학습 구조
- **글로벌-로컬 모델 결합**: 중앙집중식 글로벌 서브모델과 로컬 적응을 결합하여 non-IID 데이터 해결
- **데이터 프라이버시 보장**: 각 설계 하우스의 레이아웃 데이터를 공개하지 않고 공동 학습 가능
- **지식 공유 메커니즘**: 이종 지식 공유(heterogeneous knowledge sharing)로 강건한 중앙 모델 구축

## 검증 방법론
- **연합 학습 검증**: 다수 설계 하우스 시나리오를 시뮬레이션하여 분산 학습 효과 측정
- **non-IID 데이터 처리**: 각 클라이언트(설계 하우스)의 데이터 분포 차이를 반영한 평가
- **중앙집중식 학습 대비 비교**: 데이터를 모두 공유했을 때와 연합 학습 결과 비교
- **로컬 적응 효과 검증**: 글로벌 모델 단독 vs. 로컬 적응 결합 성능 비교

## 검증 지표
- **EPE 허용 범위**: 핫스팟/비핫스팟 분류 정확도 기반 (구체적 수치 논문 본문 참조)
- **사용 시뮬레이터**: ICCAD 2012 벤치마크 데이터셋 (표준 핫스팟 검출 벤치마크)
- **검증 커버리지**: 다수 클라이언트 환경, 다양한 데이터 분포 조건
- **주요 메트릭**: 핫스팟 검출률(True Positive Rate), 오탐률(False Positive Rate), 분류 정확도

## OPC 툴 구현 관련성
SKOPC의 핫스팟 검출 및 OPC 검증 모듈에 다음과 같이 활용 가능:
1. **분산 OPC 검증**: 다수 고객사 레이아웃을 프라이버시 보장하에 통합 핫스팟 검출
2. **전이 학습 기반 검증**: 연합 학습으로 사전 학습된 모델을 특정 공정 노드에 빠르게 적응
3. **데이터 부족 문제 해결**: 단일 설계 하우스의 제한된 핫스팟 데이터를 연합 학습으로 보완
4. **OPC 모델 신뢰성**: 다양한 레이아웃 패턴에서 학습된 핫스팟 검출기의 일반화 능력 향상
5. **협력적 OPC 개선**: 여러 파운드리/설계 하우스가 공동으로 OPC 검증 모델 개선
